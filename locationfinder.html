LocationFinder

Der LocationFinder ist eine schnelle und benutzerfreundliche interaktive Suchmaschine für Ihre Adress- und Ortsdaten, mit zahlreichen Konfigurations­möglichkeiten zur Feinabstimmung des Sucherlebnisses für den Endbenutzer.

The LocationFinder is a fast and friendly interactive search engine for your address and place data. It provides plentiful configuration possibilities for fine-tuning the end user's search experience.

Schedule a Demo Demo buchen

Every second counts when you're searching for a place, whether it's an address, a remote plot number, or a crucial point of interest. Dira GeoSystems' LocationFinder delivers results at lightning speed, helping you find what you need without delays or frustration.

LocationFinder gives you the power to search interactively through a single text field. Need to locate a specific glacier? A hydrant? A postal code? It's all at your fingertips. And because LocationFinder is an on-premise solution, your sensitive data stays secure - no compromises, just seamless performance.

Your own data

The LocationFinder operates on your own data and provides access to an interactive search, for example, to correctly position a map. Input data can come from Esri Feature Classes or a simple XML format.

Features

Funktionen

  • Interactive geocoding (given name/address, find coordinates)
  • Lexicographically sorted result list, even for partial queries
  • Reverse geocoding (given point, find nearest address/place name)
  • Very high speed, even with a large number of locations
  • Simple REST interface for queries
  • Queries can narrow down search results using Boolean filters
  • Developed in 100% .NET, no additional dependencies
  • Operates on-premise, making it suitable for sensitive data
  • Easy installation and administration
  • Optional integration into the ArcGIS platform (ArcGIS Online, ArcGIS Enterprise, ArcGIS Pro)
  • Optional integration into QGIS through a QGIS-Plugin
